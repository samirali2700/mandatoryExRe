NodeJs

Nodejs is a javascript enviromnment, javascript used to only be available on browsers (frontend),because javascript is such an easy an usefull language nodejs were developed to run javascript outside browsers, meaning javascript could be used outside browser frontend.

Nodejs uses the same engine as Chrome Browser an engine called V8 Engine

Besides coding javascript outside browsers, nodejs via npm (node package management) opens a world filled with third party packages that are easy integrated and used with ones application.
An example of this is Web Framework Express which is installed through npm

Npm

Nodejs can be downloaded and installed from browser or from third party package manager. When nodejs is installed, npm will be available globally. Npm can be used to install packages to a project that are dependent on some packages, these packages are defined in package.json
