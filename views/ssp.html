Serving Static Pages

With Express there are alot of options on how or what to return to a request, as seen in Express section of the website, a file can be returned with the function app.sendFile('Path') or in my case the files can be retrieved on the server and the send raw text that are translated to html, in this way, multiple pages or sections of a page can be combined to produce a single page

Example

A single page will most likely always include 4 sections

Snippet of each section:

Head

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="/Global.css">
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
</head>

Nav

<nav>
<ul class="w3-ul w3-bar w3-green">
<li class="w3-bar-item"><a href="/">Express</a></li>
<li class="w3-bar-item"><a href="/ssp">Serving Static Pages</a></li>
<li class="w3-bar-item"><a href="/nodejs">NodeJs</a></li>
</ul>
</nav>

Body

<div class="main">
<!-- This main div is only defined for styling purposes -every page will include a div with class 'main'-->
</div>

Footer

<footer class="w3-green w3-cell-row">
<p class="w3-cell w3-cell-middle">© 2020</p>
class="w3-cell w3-cell-middle w3-right">Styling done with w3CSS</p>
</footer>
</html>

The Sections shown above each has it's own file, the first file/section which is 'Head', starts with a <!DOCTYPE html> and opening <html> tag, notice there are no closing tag, that is because this part is only the top part of a whole page that is to be put together piece by piece. The sections can be used as template page for the website, every section will be easy and fast to change since its only one file instead of all pages

All Pages are located in folder views on root level, and the sections (Head, Nav and Footer) are located in folder views/partials

If you notice in the Head section, the first stylesheet href is pointing to '/Global.css', that is only possible because 'Global.css' is located on a public folder, which is accessible on the frontend, in fact, this website is mostly CSR (Client Side Rendering), exept the raw html that are send back with a request the styling and javascripts are first implemented when reaching the client.

Serve Page with Express

In This section, i will demonstrate how to put sections together piece by piece, and easly responde with a main page

//Firstly 'fs' is needed to retrieve the files on the server, after installing it and requiring it in main app file

const fs = require('fs');

//Then, the partials are retrieved to put together a template

const head = fs.readFileSync('./views/partials/head.html').toString();
const nav = fs.readFileSync('./views/partials/nav.html').toString();
const footer = fs.readFileSync('./views/partials/footer.html').toString();

//To make it easier and code less redundently a function is defined to return a full page, it has one parameter which is the main file i.e. the Body Section

const template = (main)=> { return head + nav + main + footer; }
